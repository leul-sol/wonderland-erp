# Wonderland Hotel brand assets

## Single logo (what you have now)

Place your file here:

```
logo.png
```

That one file is used for:

- Login and change-password screens
- Sidebar (expanded and collapsed)
- Browser tab favicon

No separate mark file is required.

## Optional `.env` overrides

```env
PORTAL_LOGO_URL=/images/brand/logo.png
PORTAL_BRAND_NAME="Wonderland Hotel"
PORTAL_PRODUCT_NAME="Wonderland ERP"
PORTAL_BRAND_TAGLINE="Hospitality operations portal"
```

If you later add a square icon-only file, set `PORTAL_LOGO_MARK_URL` for the collapsed sidebar.
